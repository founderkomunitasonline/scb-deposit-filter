Cara Kerja Filter Nominal & Detail History

  • 1. Upload Data - Upload file Manual Deposit, QRPay, dan Withdraw History
  • 2. Set Keywords - Pilih kata kunci bonus yang muncul di kolom "To Bank"
  • 3. Apply Filters - Atur nominal dan tanggal (opsional)
  • 4. Smart Filter - Sistem mengambil record TERBARU per user. Jika nominal terbaru di luar range, user TIDAK ditampilkan
  • 5. Detail History - Klik User Name untuk melihat semua riwayat deposit user (Manual + QRPay)
  • 6. Export - Download hasil ke Excel
    • 📌 Contoh: User deposit 50 (lama) dan 30 (baru), filter 50-90 → user TIDAK muncul karena record terbaru (30) di luar range filter.
